# PathFinder – Career Guidance Application

PathFinder is a web-based career guidance platform that helps students
identify suitable career paths based on their interests, skills, and personality.

## Features
- Career recommendations
- Career roadmaps
- Courses & resources
- Resume builder
- AI career chatbot

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express
- Database: MongoDB
